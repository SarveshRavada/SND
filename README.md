# Tractor Shop Modern Website

This is a mobile-first project for `SLV Tractor Spare Parts & Service Center`.

## Structure

- `index.html` - main page
- `css/style.css` - styling
- `js/script.js` - shop status and map button logic
- `assets/workshop/` - workshop images
- `assets/spare-parts/` - spare parts images

## Quick customization

1. Update phone numbers in `index.html`.
2. Add your own images into:
   - `assets/workshop/`
   - `assets/spare-parts/`
3. Update address text in `index.html`.
4. Update map coordinates in `js/script.js` if needed.
